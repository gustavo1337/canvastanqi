Descrição

O objetivo do jogo é defender uma área demarcada com a cor azul contra pequenos blindados dispostos a invadi-la. Você controla um tanque com um canhão que dispara, acerte seus inimigos e os detenha !


Mas tome cuidado ! Não encoste nos inimigos e não os deixe invadir a área demarcada com a cor azul. Se isso acontecer suas 5 vidas irão diminuir até atingirem o 0. Ai é GAME-OVER !


Como controlo meu tanque?

Para controlar o tanque você deve seguir as instruções abaixo: (OBS: Se você for jogar sozinho siga as instruções para o Jogador 1).

Jogador 1
W: Movimenta para cima
S: Movimenta para baixo
A: Movimenta para esquerda
D: Movimenta para direita
ESPAÇO: Dispara
ESC: Pausa o jogo
Jogador 2
Seta para cima: Movimenta para cima
Seta para baixo: Movimenta para baixo
Seta para esquerda: Movimenta para esquerda
Seta para direita: Movimenta para direita
ENTER: Dispara
ESC: Pausa o jogo

Sobre o jogo

Como foi criado o jogo ?

O jogo foi criado utilizando alguns conhecimentos adquiridos durante o nosso curso, principalmente em HTML, CSS, JavaScript, Canvas e Editor de Imagens em Computação Gráfica. Utilizamos também alguns tutoriais da internet que estarão disponiveis na aba créditos.


Como foram criados os elementos ?

Os elementos do jogo foram criados da seguinte maneira:


  • Interfaces: Foram codificadas em HTML e estilizadas em CSS. Todas foram criadas em uma só página e a alternância(Apenas uma é exibida enquanto outras são escondidas, ao clicar em um botão do menu principal) entre elas é feita por um script em jQuery(Framework do JavaScript). O mesmo esquema ocorre com o botão HOME que quando clicado recarrega a página mostrando apenas o menu-principal.

  • Animações: O jogo foi desenvolvido dentro do nosso elemento do HTML, o Canvas(Canvas é um elemento adicionado na versão 5 do HTML utilizado para criar desenhos, animações via JavaScript). Os veiculos, pontuação, os projéteis e as animações foram feitas no próprio Canvas utilizando JavaScript OO.
